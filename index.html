how does it work?

It's basically a cipher. A book or other stable body of text is loaded into the app, and each word is given a code in the order that it is received that coresponds to a special character (part of utf8). This code is then scrambled by the source text itself. It is impossible to read your messages without knowing the source of the cipher. Future version will allow multiple cipher sources.

The source of the cipher needs to be secret, something shared by the user and anybody who wants to read the messages, but not shared by anyone else. Since they are text files, they can exist on your computer or phone without calling attention. Keep them secret.

Once the source file is loaded, it will be cached by the app, so you can delete the file, if the situation demands it.
